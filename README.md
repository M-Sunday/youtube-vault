# YouTube Vault

Save, organize, and manage your favorite YouTube videos, bookmarks, and notes in a local app.

## Features

- **YouTube videos** — Paste a link, load metadata, organize into folders
- **Bookmarks** — Save URLs with auto-fetched preview images
- **Notes** — Rich text notes with image paste support, assignable to folders
- **Grid view** — Browse all your saved content in a visual grid
- **Folders** — Drag-and-drop organization with color coding
- **Themes** — White, Black, and Obsidian Black themes
- **Search** — Filter your vault by title

## Usage

1. Paste a YouTube link in the top bar and click the arrow to load video info
2. Click "Add video" to save it to a folder
3. Use the sidebar to browse folders, videos, bookmarks, and notes
4. Click the grid icon to switch to grid view
5. Right-click items for context menu actions (rename, delete, move, pin)

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Z` | Undo in note editor |
| `Cmd+Shift+Z` | Redo in note editor |

## Data

All data is stored locally in your browser's localStorage. Nothing is sent to any server.
